## [0.0.0] - 2026-01-15

### Initial Release

#### Added

**Core Features**:
- User authentication (login, signup, guest access)
- Kanban board with task management
- Task creation and editing
- Drag-and-drop task movement between columns
- Task priority levels (Urgent, Medium, Low)
- Task types (User Story, Technical Task)
- Subtask management
- Contact management system
- Team member assignment to tasks
- Dashboard with task statistics
- Real-time data synchronization via Firebase

**User Interface**:
- Responsive design (mobile, tablet, desktop)
- Material Design components
- Dark-friendly color scheme
- Intuitive navigation
- Task preview/detail view
- Contact detail pages
- Help documentation pages
- Privacy policy and legal notice pages

**Technical Features**:
- Built with Angular 20
- TypeScript for type safety
- Firestore real-time database
- Firebase authentication
- Firebase hosting
- RxJS for reactive programming
- Angular Signals for state management
- Angular Material for UI components
- SCSS for styling

**Database**:
- contacts collection
- tasks collection with subtasks and assignments
- appSettings collection
- Real-time synchronization

**Documentation**:
- Complete setup guide
- Architecture documentation
- API reference with examples
- Component documentation
- Database schema documentation
- Deployment guide
- Testing guide
- Troubleshooting guide
- Contributing guidelines

**Development**:
- Angular CLI configuration
- TypeScript configuration
- Firebase configuration
- Unit testing with Jasmine/Karma
- Code style guide
- Commit message conventions
